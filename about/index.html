About DigMusic

The DigMusic presence on GitHub is the work of The Music Tech Team whose areas of interest centre around audio, sound creation, and music. Software interests cover Pro Tools, Reason, Ableton Live, and Sibelius for music production, whilst Max/MSP, Pure Data, and ChucK feature for music programming.

Site-specific information

The GihHub site was written using Kramdown for Jekyll and the Notepad Jekyll theme. Workshops were written using org-mode markdown with the ReadTheOrg theme. Editing, additional programming, and any housekeeping were done using GNU Emacs and its eLISP implementation language.
